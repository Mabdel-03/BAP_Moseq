# Offspring Figures

## Overview

This directory contains figures generated from the MoSeq analysis of offspring mice from the March 2025 run.

## Directory Structure

```
figs/
└── Scalar_Figs/
    └── Position/
        └── [Position metric visualizations]
```

## Scalar Figures

### Position Metrics
Visualizations of spatial positioning metrics across environmental conditions:
- Distance to center comparisons
- Position distribution analyses
- Group-level statistical visualizations

## Figure Types

### Comparison Plots
- Pairwise condition comparisons
- Statistical significance annotations
- Sex-stratified presentations

### Distribution Plots
- Violin plots showing metric distributions
- Box plots with individual data points
- Density estimates per condition

## Environmental Conditions

Figures compare behavioral metrics across:
- **EE**: Enriched Environment
- **LNB**: Limited Nesting and Bedding
- **NGH**: Normal Growth Housing (control)
- **SI**: Social Isolation

## Generation

Figures are generated by notebooks in `../code/Analysis/`:
- `Analyze_Moseq.ipynb`: Core visualizations
- Sex-specific classifier notebooks: Classification figures

## File Formats

- **PNG**: Raster format for presentations
- **PDF**: Vector format for publication (when available)

## Related Directories

- `../code/Analysis/`: Notebooks generating figures
- `../data/`: Source data for visualizations
