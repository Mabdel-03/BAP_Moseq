# Maternal Mice Code

## Overview

This directory contains code for processing maternal mice through the MoSeq pipeline.

## Directory Structure

```
code/
└── Processing/          # MoSeq pipeline notebooks
```

## Processing

The `Processing/` directory contains Jupyter notebooks for running the MoSeq2 pipeline on maternal mice data. The population-specific model optimizes feature extraction for maternal behavioral repertoires.

## Related Directories

- `../../All/code/`: Combined population processing
- `../../Offsprings/code/`: Offspring processing with extended analysis notebooks
