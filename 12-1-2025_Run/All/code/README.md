# All Subjects Code

## Overview

This directory contains code for processing the combined population (all maternal and offspring mice) through the MoSeq pipeline.

## Directory Structure

```
code/
└── Processing/          # MoSeq pipeline notebooks
```

## Processing

The `Processing/` directory contains Jupyter notebooks for running the standard MoSeq2 pipeline on the combined dataset. This includes extraction, modeling, and visualization steps.

## Related Directories

- `../Moms/code/`: Maternal-specific processing
- `../Offsprings/code/`: Offspring-specific processing with additional analysis notebooks
