# Offspring Figures

## Overview

This directory contains figures generated from the MoSeq analysis of offspring mice, organized by analysis type and sex.

## Directory Structure

```
figs/
├── Scalar_Figs/
│   └── Position/
│       ├── Females/
│       │   └── Metric_pairwise.png
│       └── Males/
│           └── Metric_pairwise.png
└── Syllable_Usage/
    ├── Females/
    │   ├── syllable_pairwise_fdr.png
    │   └── syllable_pairwise_uncorrected.png
    └── Males/
        ├── syllable_pairwise_fdr.png
        └── syllable_pairwise_uncorrected.png
```

## Scalar Figures

### Position Metrics
Pairwise comparisons of spatial positioning metrics between environmental conditions.

**Metric_pairwise.png**
- Comparison of distance-to-center and related position metrics
- Statistical significance indicated for condition pairs
- Separate visualizations for males and females

## Syllable Usage Figures

### syllable_pairwise_fdr.png
Pairwise syllable usage comparisons with False Discovery Rate correction:
- Multiple testing correction applied
- Conservative significance threshold
- Reduces false positive findings

### syllable_pairwise_uncorrected.png
Pairwise syllable usage comparisons without correction:
- Raw p-values displayed
- More sensitive but higher false positive rate
- Useful for exploratory analysis

## Interpretation

### Environmental Condition Comparisons
Figures compare syllable usage and scalar metrics across:
- EE vs NGH (Enriched vs Control)
- LNB vs NGH (Limited Bedding vs Control)
- SI vs NGH (Social Isolation vs Control)
- EE vs LNB, EE vs SI, LNB vs SI

### Sex Stratification
All analyses are stratified by sex due to:
- Known sex differences in mouse behavior
- Optimal classification requiring sex-specific models
- Different optimal PC counts (Males: 25, Females: 12)

## Figure Generation

Figures are generated by notebooks in `../code/Analysis/`:
- `Analyze_Moseq.ipynb`: Core analysis visualizations
- Sex-specific classifier notebooks: Classification-related figures
